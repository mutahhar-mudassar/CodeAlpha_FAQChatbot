# CodeAlpha_FAQChatbot
# FAQ Chatbot — NLP Powered

> An intelligent FAQ chatbot using NLP techniques built 
> during AI Internship at CodeAlpha.

## 🧠 How It Works

Built a complete NLP pipeline from scratch in JavaScript:

1. Tokenization — splits input into words
2. Stop word removal — filters noise words  
3. Stemming — reduces words to root form
4. TF-IDF vectorization — converts text to numbers
5. Cosine Similarity — finds the best matching FAQ

No external NLP libraries used. Pure JavaScript.

## ✨ Features

- 3 FAQ domains: E-Commerce, Tech Support, University
- Real-time confidence score with visual bar
- NLP pipeline trace — see exactly how your query 
  was processed
- Related question suggestions
- Typing animation and smooth UI

## 🛠️ Tech Used

- HTML5, CSS3, Vanilla JavaScript
- NLP: Tokenization, Stop-word removal, 
  Stemming, TF-IDF, Cosine Similarity

## ▶️ How to Run

Just open faq_chatbot.html in any browser.
No installation or API key needed!
